# Contributing

Contributions are welcome if they improve clarity, security, reproducibility or documentation quality.

Rules:

1. Do not commit real passwords, private keys, API tokens or DNS provider tokens.
2. Keep example domains under `ernu.sec`.
3. Keep public author attribution: Anatolie Ernu, IT & Security Solutions, https://www.ernu.eu.
4. Test scripts on Debian 13 before submitting changes.
